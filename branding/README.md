# Conta de Casal — Brand Kit

> Tudo que você precisa para manter a identidade visual e a comunicação do **Conta de Casal** consistentes.

## Índice

| Arquivo | Descrição |
|---|---|
| [`brand-guidelines.md`](./brand-guidelines.md) | Identidade de marca, tom de voz, logotipo, paleta e regras de uso |
| [`design-system.md`](./design-system.md) | Design tokens completos, tipografia, espaçamento, componentes |
| [`design-tokens.ts`](./design-tokens.ts) | Tokens exportáveis em TypeScript para uso direto no código |
| [`marketing-strategy.md`](./marketing-strategy.md) | Estratégia de lançamento, canais, funil e orçamento |
| [`landing-page.html`](./landing-page.html) | Página institucional completa (self-contained HTML) |

## Início rápido

1. **Designers** → comece por `brand-guidelines.md`
2. **Desenvolvedores** → comece por `design-system.md` e `design-tokens.ts`
3. **Marketing** → comece por `marketing-strategy.md`
4. **Todos** → abra `landing-page.html` no navegador para ver a página institucional

## Princípios da marca

- **Empatia** → Design suave, linguagem acolhedora
- **Clareza** → Informação financeira sem complicação
- **Diversão** → Gatinhos animados e interações lúdicas
- **Confiança** → Segurança, transparência e respeito à privacidade
