# Templates (placeholders)

Esta pasta guarda modelos reutilizáveis para comunicação de marketing.

> Objetivo: acelerar execução (não é “design final”).

## Estrutura

- `brand/templates/social/` — posts, stories, anúncios estáticos
- `brand/templates/email/` — emails transacionais e de marketing
- `brand/templates/ads/` — variações de criativos + copies
