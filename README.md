# Conta de Casal

Aplicativo de gestão de despesas compartilhadas para casais.

## Stack

- **Frontend:** React + TypeScript + Vite
- **UI:** shadcn/ui + Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Edge Functions)

## Desenvolvimento

```sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build de produção
npm run build
```

## Funcionalidades

- Autenticação por PIN
- Gestão de despesas com categorias
- Divisão flexível (50/50, porcentagem, valor fixo)
- Suporte a cartões de crédito e parcelas
- Acordos recorrentes
- Estatísticas e gráficos
- Acerto de contas entre o casal
